\begin{DoxyParams}{Parameters}

This function processes an array of data to produce a CRC result.

The data in the array pointed to be {\itshape pui32\+Data\+In} is either an array of bytes or an array or words depending on the selection of the input data size options {\bfseries{CRC\+\_\+\+CFG\+\_\+\+SIZE\+\_\+8\+BIT}} and {\bfseries{CRC\+\_\+\+CFG\+\_\+\+SIZE\+\_\+32\+BIT}}.

This function returns either the unmodified CRC result or the post-\/ processed CRC result from the EC module. The post-\/processing options are selectable through {\bfseries{CRC\+\_\+\+CFG\+\_\+\+RESINV}} and {\bfseries{CRC\+\_\+\+CFG\+\_\+\+OBR}} parameters.

Write data into the EC module for CRC operations.


\begin{DoxyParams}{Parameters}
{\em ui32\+Base} & is the base address of the EC module. \\
\hline
{\em ui32\+Data} & is the data to be written.\\
\hline
\end{DoxyParams}
This function writes either 8 or 32 bits of data into the EC module for CRC operations. The distinction between 8 and 32 bits of data is made when the {\bfseries{CRC\+\_\+\+CFG\+\_\+\+SIZE\+\_\+8\+BIT}} or {\bfseries{CRC\+\_\+\+CFG\+\_\+\+SIZE\+\_\+32\+BIT}} flag is set using the \mbox{\hyperlink{group__crc__api_ga467dc1225a0182bfcdfc24c2290e9c5b}{CRCConfig\+Set()}} function.

When writing 8 bits of data, ensure the data is in the least significant byte position. The remaining bytes should be written with zero. For example, when writing 0x\+AB, {\itshape ui32\+Data} should be 0x000000\+AB.
