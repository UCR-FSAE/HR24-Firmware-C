Configures the DES module for operation.


\begin{DoxyParams}{Parameters}
{\em ui32\+Base} & is the base address of the DES module. \\
\hline
{\em ui32\+Config} & is the configuration of the DES module.\\
\hline

This function configures the DES module for operation.

The {\itshape ui32\+Config} parameter is a bit-\/wise OR of a number of configuration flags. The valid flags are grouped below based on their function.

The direction of the operation is specified with one of the following two flags. Only one is permitted.


\begin{DoxyItemize}
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+DIR\+\_\+\+ENCRYPT}} -\/ Encryption
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+DIR\+\_\+\+DECRYPT}} -\/ Decryption
\end{DoxyItemize}

The operational mode of the DES engine is specified with one of the following flags. Only one is permitted.


\begin{DoxyItemize}
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+MODE\+\_\+\+ECB}} -\/ Electronic Codebook Mode
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+MODE\+\_\+\+CBC}} -\/ Cipher-\/\+Block Chaining Mode
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+MODE\+\_\+\+CFB}} -\/ Cipher Feedback Mode
\end{DoxyItemize}

The selection of single DES or triple DES is specified with one of the following two flags. Only one is permitted.


\begin{DoxyItemize}
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+SINGLE}} -\/ Single DES
\item {\bfseries{DES\+\_\+\+CFG\+\_\+\+TRIPLE}} -\/ Triple DES
\end{DoxyItemize}
