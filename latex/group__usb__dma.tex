Assigns and configures an endpoint to a specified integrated USB DMA channel.


\begin{DoxyParams}{Parameters}
{\em ui32\+Base} & specifies the USB module base address. \\
\hline
{\em ui32\+Channel} & specifies which DMA channel to access. \\
\hline
{\em ui32\+Endpoint} & is the endpoint to assign to the USB DMA channel. \\
\hline
{\em ui32\+Config} & is used to specify the configuration of the USB DMA channel.\\
\hline
\end{DoxyParams}
This function assigns an endpoint and configures the settings for a USB DMA channel. The {\itshape ui32\+Endpoint} parameter is one of the {\bfseries{USB\+\_\+\+EP\+\_\+$\ast$}} values and the {\itshape ui32\+Channel} value is a zero-\/based index of the DMA channel to configure. The {\itshape ui32\+Config} parameter is a combination of the {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+$\ast$}} values using the following guidelines.

Use one of the following to set the DMA burst mode\+:
\begin{DoxyItemize}
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+BURST\+\_\+\+NONE}} disables bursting.
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+BURST\+\_\+4}} sets the DMA burst size to 4 words.
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+BURST\+\_\+8}} sets the DMA burst size to 8 words.
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+BURST\+\_\+16}} sets the DMA burst size to 16 words.
\end{DoxyItemize}

Use one of the following to set the DMA mode\+:
\begin{DoxyItemize}
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+MODE\+\_\+0}} is typically used when only a single packet is being sent via DMA and triggers one completion interrupt per packet.
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+MODE\+\_\+1}} is typically used when multiple packets are being sent via DMA and triggers one completion interrupt per transfer.
\end{DoxyItemize}

Use one of the following to set the direction of the transfer\+:
\begin{DoxyItemize}
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+DIR\+\_\+\+RX}} selects a DMA transfer from the endpoint to a memory location.
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+DIR\+\_\+\+TX}} selects a DMA transfer to the endpoint from a memory location.
\end{DoxyItemize}

The following two optional settings allow an application to immediately enable the DMA transfer and/or DMA interrupts when configuring the DMA channel\+:
\begin{DoxyItemize}
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+INT\+\_\+\+EN}} enables interrupts for this channel immediately so that an added call to \mbox{\hyperlink{group__usb__dma_ga77aee9e364edc04344e1fc6f71b14713}{USBDMAChannel\+Int\+Enable()}} is not necessary.
\item {\bfseries{USB\+\_\+\+DMA\+\_\+\+CFG\+\_\+\+EN}} enables the DMA channel immediately so that an added call to \mbox{\hyperlink{group__usb__dma_ga2b4d71ab71f41973357eb39e51afbdc4}{USBDMAChannel\+Enable()}} is not necessary.
\end{DoxyItemize}

{\bfseries{Example\+:}} Assign channel 0 to endpoint 1 in DMA mode 0, 4 word burst, enable interrupts and immediately enable the transfer.

\begin{DoxyVerb}//! //
//! // Assign channel 0 to endpoint 1 in DMA mode 0, 4 word bursts,
//! // enable interrupts and immediately enable the transfer.
//! //
//! USBDMAChannelConfigSet(USB0_BASE, 0, USB_EP_1,
//!                        (USB_DMA_CFG_BURST_4 | USB_DMA_CFG_MODE0 |
//!                         USB_DMA_CFG_DIR_RX | USB_DMA_CFG_INT_EN |
//!                         USB_DMA_CFG_EN));
//! \end{DoxyVerb}
 \begin{DoxyVerb}\note This feature is not available on all Tiva devices.  Please
check the data sheet to determine if the USB controller has a DMA
controller or if it must use the uDMA controller for DMA transfers.

Clears the integrated USB DMA status for a specified channel.

This function clears the USB DMA channel status for the channel specified by the {\itshape ui32\+Channel} parameter. The {\itshape ui32\+Channel} value is a zero-\/based index of the USB DMA channel to query. The {\itshape ui32\+Status} parameter specifies the status bits to clear and must be the valid values that are returned from a call to the \mbox{\hyperlink{group__usb__dma_gae03dabbb5362185a713f3f312022a444}{USBDMAChannel\+Status()}} function.

{\bfseries{Example\+:}} Clear the current USB DMA status for channel 2.

\begin{DoxyVerb}//! //
//! // Clear the any pending USB DMA status for channel 2.
//! //
//! USBDMAChannelStatusClear(USB0_BASE, 2, USBDMAChannelStatus(USB0_BASE, 2));
//! \end{DoxyVerb}
 \begin{DoxyVerb}\note This feature is not available on all Tiva devices.  Please
check the data sheet to determine if the USB controller has a DMA
controller or if it must use the uDMA controller for DMA transfers.

\return None. 
\end{DoxyVerb}
