The CRC-\/16 is computed in a running fashion, meaning that the entire data block that is to have its CRC-\/16 computed does not need to be supplied all at once. If, however, the entire block of data is not available, then {\bfseries{ui16\+Crc}} should be set to 0 for the first portion of the data, and then the returned value should be passed back in as {\bfseries{ui16\+Crc}} for the next portion of the data.
