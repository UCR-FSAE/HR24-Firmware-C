Configures the timer(s).


\begin{DoxyParams}{Parameters}
{\em ui32\+Base} & is the base address of the timer module. \\
\hline
{\em ui32\+Config} & is the configuration for the timer.\\
\hline
\end{DoxyParams}
This function configures the operating mode of the timer(s). The timer module is disabled before being configured and is left in the disabled state. The timer can be configured to be a single full-\/width timer by using the {\bfseries{TIMER\+\_\+\+CFG\+\_\+$\ast$}} values or a pair of half-\/width timers using the {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+$\ast$}} and {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+B\+\_\+$\ast$}} values passed in the {\itshape ui32\+Config} parameter.

The configuration is specified in {\itshape ui32\+Config} as one of the following values\+:


\begin{DoxyItemize}
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+ONE\+\_\+\+SHOT}} -\/ Full-\/width one-\/shot timer
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+ONE\+\_\+\+SHOT\+\_\+\+UP}} -\/ Full-\/width one-\/shot timer that counts up instead of down (not available on all parts)
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+PERIODIC}} -\/ Full-\/width periodic timer
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+PERIODIC\+\_\+\+UP}} -\/ Full-\/width periodic timer that counts up instead of down (not available on all parts)
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+RTC}} -\/ Full-\/width real time clock timer
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+SPLIT\+\_\+\+PAIR}} -\/ Two half-\/width timers
\end{DoxyItemize}

When configured for a pair of half-\/width timers, each timer is separately configured. The first timer is configured by setting {\itshape ui32\+Config} to the result of a logical OR operation between one of the following values and {\itshape ui32\+Config\+:} 


\begin{DoxyItemize}
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+ONE\+\_\+\+SHOT}} -\/ Half-\/width one-\/shot timer
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+ONE\+\_\+\+SHOT\+\_\+\+UP}} -\/ Half-\/width one-\/shot timer that counts up instead of down (not available on all parts)
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+PERIODIC}} -\/ Half-\/width periodic timer
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+PERIODIC\+\_\+\+UP}} -\/ Half-\/width periodic timer that counts up instead of down (not available on all parts)
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+CAP\+\_\+\+COUNT}} -\/ Half-\/width edge count capture
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+CAP\+\_\+\+COUNT\+\_\+\+UP}} -\/ Half-\/width edge count capture that counts up instead of down (not available on all parts)
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+CAP\+\_\+\+TIME}} -\/ Half-\/width edge time capture
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+CAP\+\_\+\+TIME\+\_\+\+UP}} -\/ Half-\/width edge time capture that counts up instead of down (not available on all parts)
\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+PWM}} -\/ Half-\/width PWM output
\end{DoxyItemize}

Some Tiva devices also allow configuring an action when the timers reach their timeout. Please consult the data sheet for the part you are using to determine whether configuring actions on timers is available.

\item {\bfseries{TIMER\+\_\+\+CFG\+\_\+\+A\+\_\+\+ACT\+\_\+\+CLRTOGTO}} -\/ clear CCP immediately and then toggle it on timeout of timer A.
