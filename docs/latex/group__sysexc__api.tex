{\footnotesize\ttfamily void Sys\+Exc\+Int\+Clear (\begin{DoxyParamCaption}\item[{uint32\+\_\+t}]{ui32\+Int\+Flags }\end{DoxyParamCaption})}

Clears system exception interrupt sources.


\begin{DoxyParams}{Parameters}
{\em ui32\+Int\+Flags} & is a bit mask of the interrupt sources to be cleared.\\
\hline
\end{DoxyParams}
This function clears the specified system exception interrupt sources, so that they no longer assert. This function must be called in the interrupt handler to keep the interrupt from being recognized again immediately upon exit.

The {\itshape ui32\+Int\+Flags} parameter is the logical OR of any of the following\+:


\begin{DoxyItemize}
\item {\bfseries{SYSEXC\+\_\+\+INT\+\_\+\+FP\+\_\+\+IXC}} -\/ Floating-\/point inexact exception interrupt
\item {\bfseries{SYSEXC\+\_\+\+INT\+\_\+\+FP\+\_\+\+OFC}} -\/ Floating-\/point overflow exception interrupt
\item {\bfseries{SYSEXC\+\_\+\+INT\+\_\+\+FP\+\_\+\+UFC}} -\/ Floating-\/point underflow exception interrupt
\item {\bfseries{SYSEXC\+\_\+\+INT\+\_\+\+FP\+\_\+\+IOC}} -\/ Floating-\/point invalid operation interrupt
\item {\bfseries{SYSEXC\+\_\+\+INT\+\_\+\+FP\+\_\+\+DZC}} -\/ Floating-\/point divide by zero exception interrupt
\item {\bfseries{SYSEXC\+\_\+\+INT\+\_\+\+FP\+\_\+\+IDC}} -\/ Floating-\/point input denormal exception interrupt
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Because there is a write buffer in the Cortex-\/M processor, it may take several clock cycles before the interrupt source is actually cleared. Therefore, it is recommended that the interrupt source be cleared early in the interrupt handler (as opposed to the very last action) to avoid returning from the interrupt handler before the interrupt source is actually cleared. Failure to do so may result in the interrupt handler being immediately reentered (because the interrupt controller still sees the interrupt source asserted).
\end{DoxyNote}

Registers an interrupt handler for the system exception interrupt.


\begin{DoxyParams}{Parameters}
{\em pfn\+Handler} & is a pointer to the function to be called when the system exception interrupt occurs.\\
\hline
\end{DoxyParams}
This function places the address of the system exception interrupt handler into the interrupt vector table in SRAM. It is the interrupt handler\textquotesingle{}s responsibility to clear the interrupt source.

Unregisters the system exception interrupt handler.

This function removes the system exception interrupt handler from the vector table in SRAM. This function also masks off the system exception interrupt in the interrupt controller so that the interrupt handler is no longer called.
